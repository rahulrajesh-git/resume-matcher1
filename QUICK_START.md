# Quick Start Guide

## For Recruiters

### 1. Create a Job Form
1. Go to `http://localhost:5000/recruiter`
2. Click **"Create New Job Form"**
3. Fill in:
   - Job Title
   - Your Email
   - Upload Job Description file
4. Click **"Create Job Form"**

### 2. Share with Aspirants
1. Get the shareable link from the form details page
2. Click **"Copy Link to Clipboard"**
3. Send via email or message

### 3. Review Applications & Match Resumes
1. Go to Dashboard
2. Click **"Run Matching"** to analyze resumes
3. Click **"View Results"** to see:
   - All applicants who submitted
   - Resume rankings with match scores
   - Contact information

### 4. Delete a Job (if needed)
1. Click **"Delete"** button
2. Confirm deletion
3. All data removed

---

## For Aspirants

### 1. Receive Job Application Link
- Open link sent by recruiter
- Format: `http://server/aspirant/<job_id>`

### 2. Submit Application
1. Fill your details:
   - Full Name
   - Email Address
   - Upload Resume
2. Click **"Submit Application"**

### 3. Confirmation
- See success message
- Application submitted and stored

---

## Key Statistics

- **Recruiter Dashboard**: See all jobs created
- **Applicant Tracking**: View all submissions per job
- **Automated Matching**: AI-powered resume ranking (0-1 score)
- **No Sign-up**: Open system, no authentication needed

---

## Default URLs

- Recruiter Dashboard: `http://localhost:5000/recruiter`
- Upload Job: `http://localhost:5000/upload_job_description`
- Application Form: `http://localhost:5000/aspirant/<job_id>`

## Supported File Formats

### Job Descriptions
- .txt (Text)
- .pdf (PDF)
- .doc / .docx (Word)

### Resumes
- .txt (Text)
- .pdf (PDF)
- .doc / .docx (Word)

### Maximum File Size
- 16 MB per file
